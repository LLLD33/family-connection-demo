    部署与 GitHub

    1. 本地运行 `node server.js`，打开 `http://127.0.0.1:3000`。
    2. 如果要自动推送微信提醒，可把 `/api/cron/daily` 挂到系统计划任务或云端 cron。
    3. GitHub 仓库只要把当前目录推上去即可，后续可以接 Railway、Render 或自己的服务器。
